## MS Restaurant Website

A restaurant website built using **React** and **Tailwind CSS**, providing users with a login and registration system, a sidebar-based navigation panel, menu browsing features, and informative pages such as **Home**, **Menu**, **About**, and **Contact Us**.

## Features

- **User login** with background image transitions  
- **User registration** with form validation  
- **Sidebar navigation** with mobile toggle  
- **Home page** with introduction, popular dishes, and reservation links  
- **Menu page** with category filters (pizza, rice, burgers, drinks, ice cream)  
- **About page** with story, mission, and team information  
- **Contact Us page** with address, phone, email, table booking form, and delivery details  
- **Responsive layout** built with Tailwind CSS  
- **Routing** using React Router  

## Setup Instructions

## 1. Install dependencies
```bash
npm install
```
### 2. Start the development server
```bash
npm start
```
